# Code of Conduct

## Our pledge

reliakit is an open, welcoming project. Everyone who takes part — contributors,
maintainers, and anyone in the issues and pull requests — is expected to help
keep it a respectful and harassment-free space, regardless of background or
experience level.

## Standard

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org),
version 2.1, as its code of conduct. The full text is available at:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

In short: be considerate and constructive, assume good faith, accept feedback
gracefully, and focus on what is best for the project and the community.
Disrespectful, demeaning, or harassing behavior is not acceptable.

## Scope

This applies in all project spaces — the repository, issues, pull requests,
discussions, and any other official channel — and when someone is representing
the project in public spaces.

## Reporting

If you experience or witness unacceptable behavior, report it privately to the
maintainer at **satyakwik@gmail.com**. Reports are handled confidentially.
Project maintainers will review every report and respond as needed, and are
obligated to respect the privacy and security of the reporter.

Maintainers who do not follow or enforce this Code of Conduct in good faith may
face consequences decided by the project's other maintainers.
